# 📊 SQL Data Analytics Project

This repository contains a collection of **SQL (T-SQL)** scripts for performing end-to-end data analysis, from exploring the database structure, analyzing trends over time, to building summary reports for customers and products.

This project is a **continuation** of a previous project, the [**data-warehouse-project**](https://github.com/farhanfdlh/data-warehouse-project), where I built an ETL (Extract, Transform, Load) pipeline to produce a data warehouse with a star schema (Gold Layer). The resulting data from that data warehouse is used as the data source for the analysis in this repository.

## 🔍 Analysis Coverage

![Roadmap](docs/roadmap_analytics.png)

The SQL scripts in this repository are organized into the following analysis categories:

| Category | Purpose |
|---|---|
| **Database Exploration** | Explore the database structure, list of tables, columns, and metadata |
| **Dimensions Exploration** | Explore the structure of dimension tables |
| **Date Range Exploration** | Determine the temporal boundaries (date range) of the data |
| **Measures Exploration** | Calculate key aggregated metrics (totals, averages, etc.) |
| **Magnitude Analysis** | Quantify and group data by specific dimensions |
| **Ranking Analysis** | Rank items (products/customers) based on performance |
| **Change Over Time Analysis** | Track trends and changes in metrics over time |
| **Cumulative Analysis** | Calculate running totals and moving averages |
| **Performance Analysis** | Measure year-over-year & month-over-month performance |
| **Data Segmentation Analysis** | Group data into meaningful categories or segments |
| **Part-to-Whole Analysis** | Compare the contribution of parts to the whole |
| **Customer Report** | Summary report of key customer metrics and behaviors |
| **Product Report** | Summary report of key product metrics and performance |
